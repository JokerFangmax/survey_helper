\section{Object-Centric and Structured Physical Dynamics}

\paragraph{Section thesis.}
Object and relation structure is a key inductive bias for physical dynamics, especially when models must generalize across configurations, infer hidden causes, or support planning.

\paragraph{Planned subsections.}
\begin{itemize}
  \item Object and relation graphs as physical inductive bias.
  \item Visual interaction models: from pixels to structured dynamics.
  \item Object-oriented prediction and planning.
  \item Limits of synthetic and controlled evaluations.
\end{itemize}
